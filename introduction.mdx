# Welcome to Chettu

Chettu (చెట్టు) is Telugu for "tree" - a fitting name for a CLI tool that captures the complete structure and content of your projects.

## What is Chettu?

Chettu is a command-line tool that generates comprehensive XML representations of your project's file structure and content. It recursively scans directories, respects ignore patterns, and outputs everything in a structured XML format that's perfect for programmatic analysis.

## Quick Example

Generating a project snapshot is as simple as:

```bash
chettu -o project.xml
```

This scans your current directory and generates an XML file with:
- A visual tree representation of your file structure
- Complete contents of all non-ignored files
- Organized, parseable XML format

<Tip>
The output is automatically copied to your clipboard (up to 50KB by default), making it easy to paste into LLM conversations or other tools.
</Tip>

## Output Structure

Chettu generates clean, structured XML:

```xml
<project>
    <project_name>My Awesome Project</project_name>
    <source_tree>
        .
        ├── README.md
        ├── src
        │   └── main.go
        └── docs
            └── guide.md
    </source_tree>
    <files>
        <file path="README.md">
            # My Project
            This is the main README file.
        </file>
        <file path="src/main.go">
            package main

            import "fmt"

            func main() {
                fmt.Println("Hello, World!")
            }
        </file>
    </files>
</project>
```
